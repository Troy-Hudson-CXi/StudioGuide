# ⚙️ ASSIGN Action

The `ASSIGN` keyword creates or updates a variable with a specified value.

## 💡 Syntax

```snippet
ASSIGN variableName = value
```

## ✅ Example

```snippet
ASSIGN name = "Troy"
ASSIGN score = 95
```

## 📌 Notes

- Variables are automatically created when first assigned.
- Strings must be in double quotes.
